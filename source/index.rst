Overview
########

This documentation is my answers to `EoPL3`__ exercises. The source code can be found on `GitHub`__.

The Scheme code is written using in the ``eopl`` language provided by DrRacket 6.1.1. Please refer to `the official DrRacket documentation`__ for more details.
